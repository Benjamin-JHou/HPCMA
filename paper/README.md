## Methods Alignment

Methods text is maintained in `paper/methods/README.md` and is aligned to:

- data snapshot and provenance manifest artifacts
- SQLite schema build and QC workflow
- query regression contracts
- reproducibility gate execution
- supplementary API scope boundary (non-clinical, non-deployment)

## Reproducibility Figure Generation

The main-text reproducibility flow figure is generated by:

```bash
python3 scripts/generate_reproducibility_flow_figure.py
```

This command updates both:

- `figures/resource_reproducibility_flow.png`
- `paper/figures/resource_reproducibility_flow.png`
